Frontend Design



  • Overview on Designing with Verilog

    • Introduction to Digital VLSI design flow- Brain storming on combinational and sequential circuit design, Importance of RTL modeling-Overview of the different features of Verilog- Different modeling styles- Data flow modeling - Structural modeling - HDL coding guidelines for proper synthesizable code- Lab: Focusing on Hierarchical designing.

    Introduction to Behavioral Modeling-Blocking and Non-blocking assignment statements - Coding techniques for synchronous and asynchronous concepts. FSM coding guidelines- Lab: Designing FSM based designs with focus on addressing latency in registering outputs. Simulation and synthesis mismatches guidelines.

  • FPGA based Design using Vivado

    • Introduction, ASIC Flow, FPGA Flow, Vivado Flow, Overview of FPGA Families, FPGAs available in the market, FPGA Architecture of latest devices - 7 series FPGA - Kintex, Artix, Virtex, Zynq

    Synthesis flow in Vivado, Implementation flow in Vivado. Overview of Logical resources on FPGA, Reset Methodology, HDL coding techniques, Lab: Vivado tool flow.

    Timing Analysis – Baselining -STA - Setup and Hold analysis registered paths, Static Timing Analysis with lab demonstrating the timing closure of a design using Vivado IDE IP core, IP customization, Hardware Debug - ILA, VIO, Lab: On usability of debug core.

  • SoC based Design using Vitis Unified Software Development

    • Introduction to SoC Technology, Introduction to Zynq SoC Architecture, Block level based design using IP Integrator tool, Embedded System Design Flow using Xilinx Vitis IDE

    Lab: Creating a Simple Embedded Hardware Design – Develop a basic embedded system for the target board on Vivado and write a basic C application using Vitis IDE to run on its exported hardware. Explore various features of Zynq SoC for Hardware-Software co-design, Introduction to AXI Interconnect Standard and its various types, Direct Memory Access Controller, Timers and Interrupts.

    Lab: Using DMA on the Zynq SoC – Add and connect block RAM to the AXI port of the processing system, Use PS DMA and GIC controllers, Use the profiling tools to analyze system performance

Physical Design

ABOUT COURSE

Physical Design Training Course mainly focused on giving complete hands on experience to physical design and physical verification flow with Industry standard tools in latest tech nodes with 80% lab practice.
